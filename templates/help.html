Dashboard — Aapka Home

Poora din ek jagah dikhta hai

Dashboard pe aate hi aapko apna sara kaam ek jagah dikh jaata hai.

  • Today's Priority Card — Sabse upar report ka status hota hai. Agar aaj submit nahi kiya toh reminder aayega.

  • Follow-Ups — Jinhe aaj follow-up karna hai unki list automatically dikhti hai (Retarget leads).

  • Quick Actions — Add Lead, Claim Leads, Submit Report ke buttons seedhe yahan milte hain.

  • KPI Cards — Total Leads, Revenue, Close Rate, Wallet Balance — chaar cards mein dikh jaata hai sab kuch.

  • Status Breakdown Chart — Donut chart mein dekho kitne leads kis stage pe hain.

  • Recent Leads — Aapke 5 latest leads neeche card view mein dikhte hain.

Tip: Analytics section mobile pe collapse rehta hai. "Show Analytics" button dabao usse kholne ke liye.

My Leads

Apne leads manage karo

Yahan sirf aapke assigned leads dikhenge. Admin ke leads alag hote hain.

  • Sidebar mein My Leads click karo.

  • Search bar mein naam ya phone se lead dhundo.

  • Status dropdown se status update karo — sirf ek click, save automatically ho jaata hai.

  • Lead ke naam pe click karo → Edit Lead page khulega — wahan notes, follow-up date, sab update hota hai.

  • Filter buttons use karo (New, Contacted, Converted, etc.) to quickly sort.

Best Practice: Har lead ka status update karo aur notes mein conversation ka summary likho — isse admin bhi track kar sakta hai.

Add Lead

Naya lead kaise add karen

  • Sidebar ya top bar mein Add Lead click karo.

  • Name aur Phone Number — yeh dono zaroori hain. Phone 10 digit ka hona chahiye.

  • Source — Kahan se aaya lead? Instagram, Facebook, WhatsApp, Referral, etc.

  • Status — Shuruaat mein "New Lead" rakho.

  • Notes — Pehli baat ya background likho.

  • Submit karo. Lead turant aapki list mein aa jaayega.

Same phone number se duplicate lead nahi ban sakta — system block kar deta hai.

Lead Pool — Leads Claim Karo

Wallet se leads kharido

Admin Lead Pool mein leads daalta hai. Aap wahan se claim karke apni list mein le aate ho.

  • Sidebar mein Lead Pool click karo.

  • Pool mein kitne leads available hain woh dikhega — price bhi (per lead).

  • Claim Leads box mein kitne leads chahiye woh number daalo.

  • Total cost calculate ho jaata hai — confirm karo.

  • Aapke wallet se amount automatically deduct ho jaayega aur leads "My Leads" mein aa jaayenge.

Pool mein leads kam hain toh admin se recharge karne bolo — aur apna wallet bhi top-up rakhna zaroori hai.

Daily Report Submit Karna

Roz ka KPI bhejna

  • Sidebar mein Daily Report click karo.

  • Aaj kitne calls kiye, kitne meetings, kitne presentations diye — sab fill karo.

  • Koi bhi notes ya highlights likhna chahte ho toh Notes section mein likho.

Report roz submit karna zaroori hai. Jo log submit nahi karte unhe admin dashboard pe "missing" mein dekha ja sakta hai.

Retarget List

Follow-up wale leads

  • Lead edit karte time Follow-Up Date set karo aur status "Retarget" rakho.

  • Sidebar mein Retarget List click karo.

  • Aaj ke liye due leads sabse upar dikhenge — WhatsApp icon se directly contact karo.

Dashboard ke "Today's Priority" card mein bhi aaj ke retarget leads dikhte hain.

My Wallet

Balance aur recharge

  • Sidebar mein My Wallet click karo.

Wallet System — Poora Flow

Recharge se leads claim tak

Wallet system teen simple steps mein kaam karta hai:

💳
Step 1
UPI se payment karo
📤
Step 2
UTR submit karo app mein
✅
Step 3
Admin approve kare → balance credit

  • Wallet page pe jaao → admin ka UPI QR scan karo Google Pay / PhonePe / Paytm se.

  • Amount daalo aur pay karo.

  • Payment success hone ke baad bank app mein UTR / Reference Number copy karo (12-digit number).

  • App mein wapas aao → "Request Recharge" form → UTR aur amount daalo → Submit karo.

  • Admin notification pe aayega → verify karke approve karega.

  • Approve hone ke baad wallet balance update ho jaata hai.

  • Ab Lead Pool mein jaao aur claim karo!

UTR number sahi daalna zaroori hai — galat UTR se request reject hogi aur process delay hoga.

UTR Number Kahan Milega?

Bank app se kaise nikalen

  • Google Pay — Payment history → Transaction → "Transaction ID" copy karo.

  • PhonePe — History → Transaction Details → "UPI Transaction ID".

  • Paytm — Passbook → Transaction → "UPI Reference Number".

  • Bank Account — Net banking ya SMS mein "UTR" ya "Ref No." milega.

UTR usually 12 digit ka hota hai aur numbers ya letters ho sakte hain (e.g., 406912345678 ya KKBK2026XXXXX).
Frequently Asked Questions
Lead Pool se claim karne ke liye wallet mein kitna balance hona chahiye?
Wallet balance = (number of leads) × (price per lead). Example: 5 leads × ₹50 = ₹250 balance chahiye. Lead Pool page pe current price aur pool count dikh jaata hai.
Main app close karta hoon toh lead status save hota hai?
Haan! Status update hote hi server pe save ho jaata hai. App close karne se koi data nahi jaata. Status dropdown change karte hi auto-save hota hai (AJAX se, form submit nahi karna padta).
Password bhool gaya — kya karun?
Login page pe "Forgot Password" link hai. Ya admin se kaho — woh Approvals page se password reset kar sakta hai.
App home screen pe add kaise karun (iPhone / Android)?
iPhone: Safari mein app kholo → Share button (square with arrow) → "Add to Home Screen" → Add karo.

Android: Chrome mein app kholo → Menu (3 dots) → "Add to Home Screen" ya "Install App" → Install karo.

Home screen pe Myle ka icon aa jaayega aur app fullscreen mode mein khulega — ekdum native app jaisi feel!
Notifications kaise enable karun?
Top navbar mein bell icon (🔕) pe click karo → browser permission poochega → Allow karo. Har naye announcement pe push notification aayega.
Lead galti se delete ho gayi — wapas kaise laaun?
Admin se kaho — Sidebar mein "Recycle Bin" hai jahan se deleted leads restore ho sakti hain. Team members apne aap restore nahi kar sakte.
Duplicate lead add karne ki koshish karun toh kya hoga?
System same phone number pe duplicate lead nahi banane deta. Error message aayega "Phone number already exists". Original lead dhundho aur update karo.
Dark Mode kaise on karun?
Top navbar mein moon icon 🌙 pe click karo — dark mode on ho jaayega. Dobara click karo toh off.
Seat Hold aur Fully Converted mein kya fark hai?
Seat Hold Confirmed — Lead ne advance/token amount de di (e.g., ₹2,000 for Slow Track). Seat confirm hai lekin full payment baaki hai.

Fully Converted — Lead ne poori payment de di. Woh formally Myle Community ka member ban gaya!
Wallet recharge request submit ki lekin admin ne approve nahi kiya — kya karun?
Admin ko directly inform karo WhatsApp pe ya personally. App mein notification aayi hogi unhe. Agar UTR sahi hai toh jaldi approve hoga. Wallet page pe request ka status dekh sakte ho.
